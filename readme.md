**WSLI**

---

WSLI stands for "WSL Improved", or "Windows Subsystem for Linux Improved" is a clean wrapper for WSL,
that is focussed on providing a clean, user-friendly WSL experience.

It has command aliases like:

```-li``` stands for list installed,
```-la``` stands for list all,
and ```default``` sets the default distro.
---
**Easier versioning:**

WSLI automatically detects if the distribution has multiple version,
like OpenSUSE, Ubuntu or Fedora. After choosing a versioned distro, 
you will be prompted with a version select screen, just type the version name as shown on screen, and you're done.
